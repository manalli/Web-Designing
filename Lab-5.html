Concept of WWW
  • It stands for World Wide Web.
  • It is basically a system of internet servers that supports specially formatted documents.

  • User acces the World Wide Web facilities via client called web browser.
HTTP Protocol
  1. HTTP stands for Hypertext Transfer Protocol.
  2. HTTP is based on the client-server arcitecture model and a stateless request/response protocol.
  3. 404 File Not Found is a common HTTP status code.
